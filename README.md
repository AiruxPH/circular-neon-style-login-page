# 🌌 Cyberpunk Neon Circular Login

A high-fidelity, futuristic circular login form built with **Vanilla HTML, CSS, and JavaScript**. Featuring orbital animations, glowing neon aesthetics, and a unique Cyberpunk atmosphere.

![Cyberpunk Login Preview](sample_screenshot.jpeg)

## ✨ Features

*   **Circular UI**: A unique, centered circular form factor.
*   **Orbital Animations**: 
    *   **Clockwise Orbit**: The login button revolves around the form.
    *   **Counter-Clockwise Orbit**: Author signature curves and revolves in the opposite direction.
*   **Cyberpunk Aesthetics**:
    *   Neon green (#61ff66) accents and glows.
    *   Flickering title animation (Night City style).
    *   Wandering background geometric objects for atmospheric depth.
*   **Modern Interactions**:
    *   Floating input labels.
    *   Custom password visibility toggle (Eye icon).
    *   Hover-pause logic for focus on interactive elements.
*   **Responsive**: Scaled for mobile devices (down to 320px).
*   **Lightweight**: Zero external JS dependencies (Vanilla JS).

## 🚀 Quick Start

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/cyberpunk-login.git
    ```
2.  **Open `index.html`**:
    Simply open the file in any modern web browser to see the animation in action.

## 🛠️ Technology Stack

*   **Structure**: HTML5
*   **Styling**: Vanilla CSS3 (Custom Variables, Keyframes, SVGs)
*   **Logic**: Vanilla JavaScript
*   **Fonts**: [Orbitron](https://fonts.google.com/specimen/Orbitron) & [Rajdhani](https://fonts.google.com/specimen/Rajdhani) via Google Fonts.
*   **Icons**: FontAwesome 6.0
